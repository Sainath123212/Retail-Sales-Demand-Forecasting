# Retail Sales Demand Forecasting

## Project Overview

This project develops a machine learning model to predict retail product demand using historical sales data. The project includes data preprocessing, exploratory data analysis, feature engineering, machine learning, model evaluation, and business recommendations.

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Dataset

- Records: 2800
- Features: 10
- Target Variable: Units Sold

## Machine Learning Model

Random Forest Regressor

## Evaluation Metrics

- MAE: 9.109
- RMSE: 10.62
- R² Score: 0.063

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Machine Learning
6. Model Evaluation
7. Business Recommendations
